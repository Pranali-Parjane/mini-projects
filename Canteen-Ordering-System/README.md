# 🍽️ College Canteen Ordering System

A web-based College Canteen Ordering System that allows students to browse the menu, place food orders online, and enables the canteen admin to manage orders efficiently.

## ✨ Features
- 🍕 Browse food menu
- 🔍 Search food items
- 🛒 Add to cart & place orders
- 📋 View order history
- 👨‍🍳 Admin dashboard
- 📊 Order & revenue management
- 🔥 Firebase Authentication & Database

## 🛠️ Tech Stack
- HTML
- CSS
- Bootstrap
- JavaScript
- Firebase Authentication
- Firebase Realtime Database
- Firebase Hosting



## 🚀 How to Run
1. Clone the repository.
2. Open the project in VS Code.
3. Configure Firebase credentials.
4. Run using Live Server.
